刚才已经我已经测试成功，并认真检查了用我的方法让wordpress的链接按左栏的链接目录里面设定的形式，只在博客首页显毫示无BUG地成功了。下面我用图片和文字重复一下具体操作，让你即使不会PHP也能这样地改好任意的一个wordpress模板，实现友情链接只在首页显示的功能。首先，我们进入后台，找到左侧的“外观”栏，点击“编辑”

编辑wordpress的模板

进入编辑页面后，观察右侧，有一个sidebar.php，也就是边栏模板。

当前主题的边栏模板

点击它后，左边的框内会出现PHP代码。我们全选这些代码，复制下来，粘贴到一个文本文档里面做好备份，如果待会友情链接没显示，却把整个页面弄烂了，也好能还原。

然后我们搜索

<?php if ( !function_exists(‘dynamic_sidebar’) || !dynamic_sidebar() ) : ?>

如果搜索不到这一句，搜“dynamic_sidebar”即可，然后顺着它，一直找到“endif;”（PHP代码均无引号）
